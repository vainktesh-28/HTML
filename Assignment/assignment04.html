All HTML tags

  1. Document Structure

    • <'!DOCTYPE html'> : Declares the document type and version of HTML
    • <'html'> : Root element of the HTML document.
    • <'head'> : Contains meta-information about the document, such as title and links to stylesheets.
    • <'title'> : Sets the title of the document (displayed in the browser tab).
    • <'meta'> : Defines metadata like character set, viewport settings, etc.
    • <'link'> : Links to external resources like stylesheets.
    • <'style'> : Contains internal CSS styles.
    • <'body'> : Contains the content of the document, including text, images, and other elements.

  2. Text formatting

    • <'h1'> to <'h6'> : Headings, with <'h1'> being the largest and <'h6'> the smallest.
    • <'p'> : Paragraph.
    • <'br'> : Line break.
    • <'hr'> : Horizontal rule (line).
    • <'strong'> : Defines important text (bold).
    • <'em'> : Defines emphasized text (italic).
    • <'b'> : Bold text (non-semantic).
    • <'i'> : Italic text (non-semantic).
    • <'u'> : Underlined text.
    • <'mark'> : Highlighted text.
    • <'small'> : Smaller text
    • <'del'> : Deleted text.
    • <'ins'> : Inserted text.

  3. Lists

    • <'ul'> : Unordered list.
    • <'ol'> : Ordered list.
    • <'li'> : List item.

  4. Links and anchors

    • <'a'> : Defines a hyperlink.
    • <'link'> : Links to external resources (usually used in the ).
    • <'nav'> : Navigation links.

  5. Images and multimedia

    • <'img'> : Embeds an image.
    • <'map'> : Defines an image map
    • <'area'> : Defines a clickable area within an image map.
    • <'audio'> : Embeds audio content.
    • <'video'> : Embeds video content.
    • <'source'> : Specifies multiple media resources for <'audio'> and <'video'>.
    • <'track'> : Specifies text tracks for <'video'> and <'audio'> elements.

  6. Tables

    • <'table'>: Defines a table.
    • <'tr'> : Defines a row in a table
    • <'td'> : Defines a cell in a table row.
    • <'th'> : Defines a header cell in a table.
    • <'caption'> : Defines a table caption.
    • <'thead'> : Grouping of header content in a table.
    • <'tbody'> : Grouping of body content in a table.
    • <'tfoot'> : Grouping of footer content in a table.

  7. Forms and input elements

    • <'form'> : Defines an HTML form.
    • <'input'> : Defines an input control.
    • <'textarea'> : Defines a multi-line text input control
    • <'button'> : Defines a clickable button.
    • <'select'> : Defines a drop-down list
    • <'option'> : Defines options in a drop-down list.
    • <'label'> : Defines a label for an <'input'> element.
    • <'fieldset'> : Groups related elements in a form.
    • <'legend'> : Defines a caption for the <'fieldset'> element.

  8. Semantic elements

    • <'header'> : Defines a header for a document or section.
    • <'nav'> : Defines navigation links.
    • <'main'> : Defines the main content of the document.
    • <'section'> : Defines a section in a document.
    • <'article'> : Defines an article.
    • <'aside'> : Defines content aside from the main content.
    • <'footer'> : Defines a footer for a document or section.
    • <'figure'> : Specifies self-contained content, such as images, diagrams, or code.
    • <'figcaption'> : Provides a caption for the <'figure'> element.

  9. Scripting

    • <'script'> : Defines client-side JavaScript.
    • <'noscript'> : Defines alternative content for users who have disabled JavaScript.

  10. Embeded content

    • <'iframe'> : Defines an inline frame.
    • <'object'> : Defines an embedded object.
    • <'embed'> : Defines a container for external interactive content.

  11. Miscellaneous

    • <'data'> : Links content with a machine-readable translation.
    • <'datalist'> : Contains a set of <'option'> elements for an <'input'> element.
    • <'keygen'>: Provides a key-pair generator field (deprecated).
    • <'output'> : Represents the result of a calculation or user action.
    • <'progress'> : Represents the progress of a task.
